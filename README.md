# 洛克王国攻略小程序

## 功能模块

### 1. 首页
- Banner轮播展示最新活动
- 快捷入口：宠物图鉴、攻略指南、活动日历、战斗模拟
- 热门宠物推荐
- 最新资讯

### 2. 宠物图鉴
- 支持按属性筛选（火、水、草、普通等18种属性）
- 支持按名称搜索
- 宠物列表展示：名称、属性、种族值等基本信息
- 点击进入宠物详情页

### 3. 宠物详情
- 宠物头像和基本信息
- 三个标签页：基本信息、种族值、技能
- 种族值可视化条形图
- 支持分享和复制信息

### 4. 攻略指南
- 分类筛选：新手、进阶、PVP、PVE、资源、捕捉
- 可展开/收起攻略内容
- 详细的游戏攻略和技巧

### 5. 活动日历
- 月度日历视图，标记有活动的日期
- 按状态筛选：进行中、即将开始、长期
- 活动卡片展示：类型、描述、奖励、时间

### 6. 战斗模拟器
- 选择攻击方和防御方宠物
- 选择技能
- 实时计算伤害
- 显示暴击效果
- 战斗日志记录
- 属性克制说明

## 技术特点

- 原生微信小程序开发
- 响应式UI设计
- 本地数据存储
- 流畅的交互动画

## 使用方法

1. 使用微信开发者工具导入项目
2. 填入你的小程序AppID
3. 编译运行即可
