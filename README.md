# 洛手助手BENJAMIN v1.0.8F

洛手助手BENJAMIN v1.0.8F，提供捕捉统计、活动日历、远行商人等攻略辅助功能。

## 功能模块

### 1. 首页
- Banner轮播展示最新活动
- 快捷入口：精灵图鉴、道具图鉴、孵蛋查询、捕捉统计、战斗模拟、攻略指南
- 公告栏展示（有公告全员可见，无公告管理员可见）
- 公告支持订阅小程序推送通知
- 热门宠物推荐

### 2. 精灵图鉴
- 支持按属性筛选（火、水、草、普通等18种属性）
- 支持按名称搜索
- 精灵列表展示：名称、属性、种族值等基本信息
- 点击进入精灵详情页

### 3. 精灵详情
- 精灵头像和基本信息
- 三个标签页：基本信息、种族值、技能
- 种族值可视化条形图
- 支持分享和复制信息

### 4. 道具图鉴
- 道具分类浏览
- 道具详情信息

### 5. 攻略指南
- 分类筛选：新手、进阶、PVP、PVE、资源、捕捉
- 可展开/收起攻略内容
- 详细的游戏攻略和技巧

### 6. 活动日历
- 月度日历视图，标记有活动的日期
- 按状态筛选：进行中、即将开始、长期
- 活动卡片展示：类型、描述、奖励、时间
- 管理员可编辑/新增/删除活动

### 7. 战斗模拟器
- 选择攻击方和防御方精灵
- 选择技能
- 实时计算伤害
- 显示暴击效果
- 战斗日志记录
- 属性克制说明

### 8. 捕捉统计
- 捕捉次数与成功率统计
- 奇遇事件统计（异色、狂欢等）
- 保底进度条动态粒子效果
- 异色统计清除选项
- 长按可输入捕捉数量，全页面关联计数

### 9. 孵蛋查询
- 孵蛋规则查询
- 蛋组信息展示

### 10. 远行商人
- 商人物品价格展示
- 支持管理员自定义编辑内容
- 支持订阅小程序推送通知

### 11. 管理后台
- 公告管理：发布、编辑、删除、置顶
- 公告和活动支持图片上传（相册/拍照）或链接输入
- 公告和活动支持导入XML/DOCX文件自动解析内容
- 用户管理：封禁、解封、设为小编
- 数据统计：总用户数、总公告数
- 页面装修：远行商人、孵蛋查询支持默认/自定义切换编辑

### 12. 关于我们
- 开发者信息
- 免费福利
- 联系方式
- 数据来源说明
- 隐私政策
- 免责声明
- 更新日志
- 管理员可在线编辑页面内容

### 13. 登录
- 微信头像和昵称授权
- 累计登录天数等级系统
- 3天登录状态有效期
- 云端数据同步

## 技术特点

- 原生微信小程序开发
- 暗色科技感主题UI
- 微信云开发（云数据库 + 云存储）
- 本地数据存储与云端同步
- 流畅的交互动画（流光特效、粒子效果）
- 管理员权限系统

## 使用方法

1. 使用微信开发者工具导入项目
2. 填入你的小程序AppID
3. 在云开发控制台创建以下数据库集合：`admin_config`、`users`、`comments`、`announcements`、`about_config`、`site_config`、`page_config`、`subscribers`
4. 设置集合权限为"所有用户可读，仅创建者可写"
5. 编译运行即可
